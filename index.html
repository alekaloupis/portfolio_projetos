Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com Streamlit

Nesse projeto os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e o GitHub, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa market place de restaurantes.

O resultado final do projeto foi um painel hospedado em ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • GitHub
